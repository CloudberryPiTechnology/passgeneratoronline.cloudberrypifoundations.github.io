# Password Generator User Guide

## Overview

Password Generator is a simple tool that creates random passwords using letters and numbers. It allows you to either display the generated password on the screen or copy it directly to your clipboard.

---

## Getting Started

1. Open the Password Generator webpage.
2. Find the **Password Length** field.
3. Enter the number of characters you want your password to contain.

### Examples

| Length Entered | Example Result |
|---------------|---------------|
| 8 | `aB7kP2mQ` |
| 12 | `K9mX2pQr7LwA` |
| 20 | `7Gh4LpQ8vNx2Tk9ZmWdR` |

---

## Generating a Password

### Generate & Show

1. Enter a password length.
2. Click **Generate & Show**.
3. The generated password will appear on the screen.

### Generate & Copy

1. Enter a password length.
2. Click **Generate & Copy**.
3. A new password will be generated and copied to your clipboard automatically.
4. Paste it anywhere using:

- **Ctrl + V** on Windows/Linux
- **Cmd + V** on macOS

---

## Password Contents

Generated passwords contain only:

- Uppercase letters (`A-Z`)
- Lowercase letters (`a-z`)
- Numbers (`0-9`)

The generator does **not** use special characters such as:

```text
! @ # $ % ^ & * ( ) _ + - = ? / \
```

This makes the passwords compatible with websites and systems that restrict special characters.

---

## Tips for Strong Passwords

For better security, consider the following recommendations:

| Purpose | Recommended Length |
|----------|------------------|
| Basic accounts | 8–12 characters |
| Most websites | 12–16 characters |
| Important accounts | 16–20 characters |
| Maximum security | 20+ characters |

Additional tips:

- Use a unique password for every account.
- Do not share your passwords.
- Consider using a password manager to store passwords securely.
- Longer passwords are generally more secure than shorter ones.

---

## Troubleshooting

### Invalid Password Length

If you enter:

- Letters
- Symbols
- Zero (`0`)
- Negative numbers

the generator will display an error and ask for a valid positive number.

### Clipboard Copy Does Not Work

Some browsers require permission to access the clipboard.

If copying fails:

1. Use **Generate & Show**.
2. Select the password manually.
3. Copy it using **Ctrl + C** or **Cmd + C**.

---

## Privacy

All password generation happens entirely within your web browser.

- No passwords are sent to a server.
- No passwords are stored.
- No personal information is collected.

Your generated passwords remain on your device.
